# Stream Cutter Beta: инструкция по установке

## Что вы получите

Для бета-теста нужен один файл:

`Stream Cutter_0.1.0_x64-setup.exe`

Внутри установщика уже находятся Stream Cutter, `yt-dlp.exe`, `N_m3u8DL-RE.exe`, `ffmpeg.exe` и `ffprobe.exe`.

## Установка

1. Скачайте `Stream Cutter_0.1.0_x64-setup.exe`.
2. Запустите файл двойным кликом.
3. Если Windows SmartScreen покажет предупреждение, нажмите `Подробнее`, затем `Выполнить в любом случае`.
4. Дождитесь окончания установки.
5. Запустите `Stream Cutter` из меню Пуск или ярлыка.

## Первая активация

При первом запуске приложение покажет экран активации.

1. Нажмите `Копировать` рядом с `Machine ID`.
2. Отправьте этот Machine ID владельцу бета-теста.
3. Получите персональный ключ.
4. Вставьте ключ в поле `Лицензионный ключ`.
5. Нажмите `Активировать`.

Ключ работает только на том компьютере, для которого был выдан Machine ID.

## Сброс ключа

Если нужно ввести новый ключ:

1. Откройте `Настройки`.
2. Найдите блок `Лицензия beta`.
3. Нажмите `Сбросить ключ`.
4. Приложение перезагрузится и снова покажет экран активации.
5. Введите новый ключ.

## Где лежат данные

Лицензия хранится локально:

`%APPDATA%\Stream Cutter\license.json`

Настройки приложения хранятся в стандартной папке данных Tauri/WebView. Очередь и логи зависят от текущих настроек приложения и выбранной папки загрузок.

## Частые проблемы

### Windows блокирует запуск

Это нормально для ранней бета-сборки без дорогой code-signing репутации. Используйте `Подробнее` -> `Выполнить в любом случае`, если файл получен от владельца бета-теста.

### Приложение просит ключ после переустановки

Обычно переустановка сохраняет `%APPDATA%\Stream Cutter\license.json`. Если файл был удалён вручную или чистильщиком системы, введите тот же ключ снова.

### Ключ не подходит

Проверьте, что вы отправили владельцу беты Machine ID именно с этого компьютера. Ключ от другого ПК будет отклонён.

### Нет WebView2

На Windows 11 WebView2 обычно уже установлен. На старых Windows 10 установите Microsoft Edge WebView2 Runtime с сайта Microsoft и запустите Stream Cutter снова.

### Антивирус ругается на ffmpeg или yt-dlp

Некоторые антивирусы настороженно относятся к консольным downloader-инструментам. Добавьте папку установки Stream Cutter в исключения только если доверяете источнику установщика.

### Не скачивает через сеть

Проверьте интернет, VPN/прокси и настройки `Настройки` -> `Сеть`. Если сервис доступен только через прокси, включите прокси и укажите адрес вида `http://127.0.0.1:2080` или `socks5://127.0.0.1:1080`.

## Для владельца беты: выпуск ключей

Один ключ:

```powershell
npm run license:keygen -- --machine MACHINE_ID_ТЕСТЕРА --name "Tester 01" --note "Discord @name"
```

Пакетно для 50 тестеров:

1. Заполните `beta-testers-50.csv`: в колонку `machine_id` вставьте Machine ID каждого тестера.
2. Запустите:

```powershell
npm run license:keygen -- --csv beta-testers-50.csv --out beta-keys.generated.csv
```

3. Раздайте каждому тестеру ключ из его строки.

Без Machine ID невозможно заранее выпустить честно привязанные оффлайн-ключи. Если сделать универсальные ключи, их можно будет использовать на разных компьютерах.
